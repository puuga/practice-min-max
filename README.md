# practice-min-max

## Question

Find min and max from input.

By given 1 interger, value between 1 and 1,000 as the next input.

And 2-n interger, value between -1,000,000 and 1,000,000.

Example

| input | output |
|---|---|
| 5 | -10 |
| 5 | 10 |
| 9 |
| -10 |
| -5 |
| 10 |
